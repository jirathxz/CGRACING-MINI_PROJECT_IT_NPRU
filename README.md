# CG RACING - E-Commerce Web Concept

โครงการพัฒนาต้นแบบเว็บไซต์พาณิชย์อิเล็กทรอนิกส์ (E-Commerce) ที่ออกแบบมาเพื่อจำหน่ายผลิตภัณฑ์ดูแลรักษารถยนต์และน้ำมันเครื่องโดยเฉพาะ ตัวเว็บไซต์เน้นการนำเสนอภาพลักษณ์ที่ทันสมัยด้วยธีมสีเข้ม (Dark Mode) และการจัดการส่วนติดต่อผู้ใช้ (User Interface) ให้มีความลื่นไหล

---

## เทคโนโลยีและเครื่องมือที่ใช้

* **HTML5:** ใช้กำหนดโครงสร้างของเนื้อหาและองค์ประกอบต่าง ๆ ภายในหน้าเว็บ
* **CSS3 (Custom Variables):** ใช้ระบบตัวแปรในการควบคุมชุดสีหลัก (Palette) เพื่อให้เกิดความสม่ำเสมอของดีไซน์ทั้งโครงการ และใช้ **Media Queries** เพื่อรองรับการแสดงผลแบบ Responsive
* **JavaScript (Vanilla JS):** ใช้สำหรับควบคุมระบบ Logic ต่าง ๆ ดังนี้:
    * **Hero Slider:** ระบบแบนเนอร์เลื่อนอัตโนมัติพร้อมการคำนวณตำแหน่งผ่าน Index
    * **Touch Events:** รองรับการปัดหน้าจอ (Swipe) สำหรับผู้ใช้งานบนอุปกรณ์มือถือ
    * **Dynamic Content:** ระบบจำลองรายการสินค้า (Product Cloning) เพื่อทดสอบการจัดวาง Layout ในสภาวะที่มีข้อมูลจำนวนมาก
* **Font Awesome:** ใช้สำหรับแสดงผลไอคอนสัญลักษณ์ในส่วนของเมนูและรายละเอียดสินค้า
* **Google Fonts:** ใช้งานแบบอักษร 'Prompt' เพื่อความเป็นสากลและรองรับการอ่านภาษาไทยได้อย่างชัดเจน

---

## คุณสมบัติเด่นของโครงการ

### 1. Interactive Sidebar Navigation
แถบเมนูด้านข้างถูกออกแบบให้เป็นแบบย่อ (Collapsed) ในสภาวะปกติเพื่อเพิ่มพื้นที่การมองเห็น และจะขยายตัว (Expand) พร้อมแสดงข้อความกำกับเมื่อมีการวางเมาส์ (Hover) โดยมีการใช้ Transition เพื่อความนุ่มนวลในการเคลื่อนไหว

### 2. Advanced Hero Banner
ระบบแบนเนอร์หน้าแรกประกอบด้วย:
* **Auto-sliding:** เลื่อนเปลี่ยนรูปภาพเองทุก 5 วินาที
* **Manual Control:** สามารถเลือกชมแบนเนอร์ที่ต้องการได้ผ่านจุดนำทาง (Navigation Dots)
* **User Interruption:** ระบบจะหยุดเลื่อนอัตโนมัติชั่วคราวเมื่อผู้ใช้งานมีการปฏิสัมพันธ์กับแบนเนอร์

### 3. Responsive Web Design
เว็บไซต์ได้รับการปรับแต่งให้รองรับการใช้งานผ่านอุปกรณ์ที่หลากหลาย:
* **Desktop:** แสดงผล Sidebar และเมนูเต็มรูปแบบ
* **Mobile (max-width: 768px):** ระบบจะทำการซ่อน Sidebar อัตโนมัติและปรับขนาด Grid ของสินค้าให้เหมาะสมกับหน้าจอขนาดเล็กเพื่อความสะดวกในการเลือกซื้อ

### 4. Categorization & Filtering
* **Categories Grid:** ในหน้าหลักมีการแบ่งหมวดหมู่ยอดนิยม เช่น น้ำมันเครื่อง, น้ำมันเฟืองท้าย และชุดเปลี่ยนถ่ายสินค้า
* **Filter System:** ในหน้าค้นหา (Discover) มีการเตรียมปุ่มตัวกรองเพื่อจำแนกประเภทสินค้าที่ต้องการเข้าถึงได้อย่างรวดเร็ว

---

## โครงสร้างไฟล์ในโครงการ

* `index.html`: หน้าหลักของเว็บไซต์ ประกอบด้วยส่วน Header, Sidebar, แบนเนอร์สไลด์ และรายการสินค้าแนะนำ
* `discover.html`: หน้าสำหรับค้นหาและเลือกดูสินค้าทั้งหมด พร้อมระบบตัวกรองประเภทสินค้า
* `style.css`: ไฟล์หลักที่ใช้ควบคุมการแสดงผล การจัดวาง (Layout) และการกำหนดธีมสีทั้งหมดของโครงการ

---

## วิธีการใช้งาน

1.  จัดเก็บไฟล์ `index.html`, `discover.html` และ `style.css` ไว้ในโฟลเดอร์เดียวกัน
2.  เปิดไฟล์ `index.html` ผ่านเว็บเบราว์เซอร์เพื่อเข้าสู่หน้าหลักของระบบ
3.  สามารถสลับไปยังหน้าค้นหาได้ผ่านทางแถบ Sidebar ในเมนู "Discover" หรือ "ค้นหา"
