# CG RACING - E-Commerce Web Concept

โครงการพัฒนาต้นแบบเว็บไซต์พาณิชย์อิเล็กทรอนิกส์ (E-Commerce) สำหรับจำหน่ายอุปกรณ์และผลิตภัณฑ์ดูแลรักษารถยนต์ โดยเน้นการออกแบบส่วนติดต่อผู้ใช้ (User Interface) ที่ทันสมัยในรูปแบบ Dark Mode เพื่อสร้างภาพลักษณ์ที่น่าเชื่อถือและเหมาะสมกับกลุ่มผู้ใช้งานสายยานยนต์

---

## เทคโนโลยีที่ใช้ในการพัฒนา

การพัฒนาโปรเจกต์นี้อาศัยเทคโนโลยีมาตรฐานเว็บ (Web Standards) เพื่อให้การแสดงผลรวดเร็วและรองรับการปรับแต่งในอนาคต:

* **HTML5:** กำหนดโครงสร้างหลักของเนื้อหา (Semantic HTML) แบ่งส่วน Header, Sidebar, Main และ Article อย่างชัดเจน
* **CSS3:** ใช้ระบบ **CSS Variables** ในการควบคุมธีมสี (Theming) และใช้งาน **Flexbox/Grid Layout** เพื่อจัดวางองค์ประกอบให้มีความยืดหยุ่น
* **JavaScript (Vanilla JS):** จัดการระบบ Logic ฝั่งหน้าบ้าน (Client-side) เช่น ระบบแบนเนอร์เลื่อนอัตโนมัติ และการจำลองข้อมูลสินค้า (Product Cloning)
* **Google Fonts:** ใช้งาน Font 'Prompt' เพื่อรองรับการอ่านภาษาไทยและภาษาอังกฤษที่ลื่นตา
* **Font Awesome:** ใช้งานไอคอนเวกเตอร์สำหรับการสื่อสารผ่านสัญลักษณ์ภายในหน้าเว็บ

## คุณสมบัติเด่นของระบบ

### 1. Adaptive Navigation Sidebar
แถบเมนูด้านข้างถูกออกแบบให้มีการตอบสนองแบบ Interactive โดยจะแสดงเฉพาะไอคอนในสภาวะเริ่มต้นเพื่อประหยัดพื้นที่ และจะขยายตัวเพื่อแสดงเมนูคำอธิบายเมื่อมีการวางเมาส์เหนือองค์ประกอบ (Hover Effect)

### 2. Intelligent Hero Slider
หน้าหลัก (Home) มาพร้อมกับระบบ Hero Banner ที่ทำงานแบบอัตโนมัติ และรองรับการควบคุมผ่านจุดนำทาง (Navigation Dots) รวมถึงการรองรับ Touch Event สำหรับการปัด (Swipe) บนอุปกรณ์เคลื่อนที่

### 3. Responsive Web Design (RWD)
โครงสร้างของเว็บไซต์ถูกออกแบบให้ปรับเปลี่ยนตามขนาดหน้าจอ (Breakpoints) โดยเฉพาะในส่วนของรายการสินค้า (Product Grid) ที่จะปรับจำนวนคอลัมน์อัตโนมัติ และการซ่อน Sidebar บนหน้าจอขนาดเล็กเพื่อประสบการณ์การใช้งานที่ดีที่สุดบนสมาร์ทโฟน

### 4. Product Catalog & Filtering
มีการจัดหมวดหมู่สินค้าไว้อย่างเป็นสัดส่วน พร้อมส่วนของหน้า Discover ที่รองรับปุ่มกรองประเภทสินค้า (Filter Buttons) เพื่อความสะดวกในการจำแนกกลุ่มผลิตภัณฑ์ เช่น น้ำมันเครื่อง หรือชุดอุปกรณ์เช็คระยะ

---

## โครงสร้างไฟล์ในโปรเจกต์

* `index.html`: หน้าแรกของเว็บไซต์ แสดงส่วนของ Banner ประชาสัมพันธ์ หมวดหมู่สินค้า และรายการสินค้าแนะนำ
* `discover.html`: หน้าสำหรับค้นหาและเลือกดูสินค้าทั้งหมด พร้อมระบบตัวกรองประเภท
* `style.css`: ไฟล์สไตล์หลักที่ควบคุมการแสดงผลและธีมสี (Global Styles)
* `fonts/`: โฟลเดอร์เก็บแบบอักษรพิเศษที่ใช้ภายในโครงการ
